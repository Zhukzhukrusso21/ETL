Для найстройки приложения перейдите в файл конфигурации проекта.

Свойство connectionstring отвечает за подключение к БД.
Базовое значение подключается к локальной базе данных localdb)\MSSQLLocalDB

Найстроки:
	#key - указывает путь к файлу .csv с данными api 
	#name - определяет имя созданной таблицы
	#port - устанавливает значение порта в WireMockServer для создание странички Api c данными стран
	#urls - устанавливает значение пути адреса в WireMockServer для создание странички Api c данными стран
	#cntTask - устанавливает макимальное количество потоков
